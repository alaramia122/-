# Анализатор частоты слов в тексте через префиксное дерево, топ-K  
**Учебный проект по дисциплине ПЯВУ**  
**Автор:** Осипенко Гавриил Васильевич  
**Группа:** М7О-407С-22  
**Год:** 2025  

---
### О проекте
Программа выполняет частотный анализ русских текстов с использованием **префиксного дерева** и выводит **Топ-K** самых частых слов.

**Особенности:**
- Полная поддержка русского языка (кириллица)
- Три режима сортировки: по частоте, по алфавиту, без сортировки
- Анализ одного файла или всей папки
- Красивый консольный интерфейс с рамками
- Автоматическое сохранение результатов в папку `Records`
  
- **Стандарт языка C++17**

---
### Требования к текстовым файлам
- Кодировка: **ANSI (Windows-1251)** — обязательно!
- Расположение: папка `SourceTexts` рядом с `.exe`
- Формат: `.txt`

> Внимание: UTF-8 не поддерживается — будет некорректное чтение русских букв.

---
### Как запустить
1. Скачайте релиз или соберите проект в Visual Studio 2022+
2. Поместите текстовые файлы в папку `SourceTexts`
3. Запустите `.exe`
4. Следуйте подсказкам на экране
